# claude-launcher

Claude のスターバーストロゴ (ユーザ表現で「ウニ」) を 2 つのボタンから打ち出して物理挙動で遊ぶ
シンプルな 2D シューター。 `claude-suika` の元になったとも言える 「Claude ロゴを物理オブジェクトとして扱う」
発想の最初の形。

## 最終形 (v3) — 2 ボタン式ランチャー

### 操作

| 入力 | 挙動 |
|---|---|
| 画面下の **↑ ボタン** | 画面下端のランダム位置から Claude を上に打ち上げる |
| 画面下の **↓ ボタン** | 画面上端のランダム位置から Claude を下に撃ち下ろす |
| ゲーム内の Claude をタップ | 11 本の棒がそれぞれランダムに伸縮 + グローハイライト |

連打でどんどん増やせる。

### 物理

- 打ち出し後は自由落下 (重力)
- Claude 同士は相互干渉 (衝突)
- 撃ち下ろし版も同様に下落・干渉

### ピース (Claude) — SVG スターバースト

- **11 本の棒** を放射状に配置
- 各棒の長さは微妙に違う (ロゴ画像に合わせた手調整)
- タップ反応:
  - 棒ごとに **ランダムに伸縮** してから元の長さに戻る
  - 押下瞬間の **グローハイライト** (v1 から継続)

## 進化ログ (要点)

| 版 | 内容 | 結果 |
|---|---|---|
| v1 | テキスト入力欄 + ↑ ボタン | 入力文字数分の Claude を下から打ち上げ。 空欄でもボタン押下で 1 つ射出。 ロゴ衝突 + タップで跳ねる |
| v2 | SVG スターバースト化 | 画像準拠の 11 本棒。 タップで各棒ランダム伸縮、 グロー演出は維持。 中央配置に調整 |
| v3 | テキスト入力撤廃 + 2 ボタン | ↑ で下から打ち上げ、 ↓ で上から撃ち下ろし。 連打スポーン |

## 設計上の学び (転用可能)

- **入力 UI の単純化** — テキスト入力 → ボタンだけ、 操作 1 種類に集約すると連射性が増して「触って遊ぶ」 体験になる
- **Claude ロゴ = 物理オブジェクト** という見立ては Ludellus の知育コンテンツ全般で再利用可能 (ピース・粒子・チャージ表現)
- **タップ反応は棒の伸縮 + グロー** が Claude ブランド表現として座りが良い (suika にも継承)
