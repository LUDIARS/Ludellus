# claude-suika

Claude ロゴをピースにしたスイカゲーム風物理マージゲーム。 ソース: [`claude-suika.jsx`](./claude-suika.jsx)

## 全体像

- 画面下のバーで照準位置を決め、 上方向の ↑ ボタン位置から Claude ロゴを落下させる
- 同色・同サイズの Claude が接触するとマージしてサイズが 1 段階上がる
- 16x が 2 つマージすると両方消えてボーナススコア (350pts)
- 画面上部の **GAME OVER LINE** (赤い破線) を Claude が越えるとゲームオーバー

## 操作

| 入力 | 挙動 |
|---|---|
| 画面下のバーをドラッグ | ↑ ボタンが左右に移動 (照準) |
| バーから指を離す | 離した瞬間に Claude を落下させる (pointerCapture で指がバー外に出ても追従) |
| ゲームエリア内の Claude をタップ | `burstLogo()` 発火 (後述) |

ドロップ後 ~0.45 秒のクールダウンあり。

## ピース (Claude) の仕様

### サイズ

- レベル 1 〜 5 (倍率としては 1x / 2x / 4x / 8x / 16x)
- マージで 1 段階アップ。 16x 同士は両方消滅 + 350pts
- **Claude 中央に倍率テキストは表示しない** (純粋にスターバースト形状のみ)

### 出現サイズ (レアサイズ)

- 80% → レベル 1 (通常)
- 20% → レベル 1〜5 のランダム (大きいのが突然降ってくることがある)

### 色 (進行に応じてアンロック)

| 落下個数 | 色数 | 追加される色 |
|---|---|---|
| 0 | 3 | オレンジ / 緑 / 青 (初期) |
| 20 | 4 | パープル |
| 40 | 5 | イエロー |
| 60 / 80 / 100 | 6+ | (未定義、 必要なら追加) |

- アンロック時にトースト通知を表示
- ヘッダーに 5 つのドットで解放状態 + `X/20` カウンタ

## マージ演出

- 接触判定が成立すると新サイズのピースに置換 + 発光フラッシュ
- 16x + 16x 消滅時も同様のフラッシュ + スコア加算

## タップでバースト (`burstLogo`)

ゲームエリアの Claude をタップした時の挙動。 視覚演出 + 物理影響あり。

### 視覚

- Claude の棒が **1.5〜2.6 倍** にランダム伸長
- `BURST_LERP = 0.18` で素早く伸び、 `BURST_RETURN_LERP = 0.09` でゆっくり戻る
- 28 フレーム後にターゲットを 1.0 にリセット → じんわり収縮

### 物理影響

- タップした Claude を中心に **半径 r × 2.4** に衝撃波 (距離 falloff)
- Claude のサイズが大きいほど爆風が強い (`r / 16` でスケール)
- 上向きにわずかなバイアス (積み重なった山を崩しやすくするため)
- タップした Claude 自身も少し上に跳ねる

## ゲームオーバー判定

- 画面上部の赤い破線 (GAME OVER LINE) を Claude が越えたら即終了
- 射出直後は **猶予フレーム** あり (積み上げ中に着地直後で即終了しないようにする)

## ヘッダー UI

- スコア表示
- **NEXT** プレビュー (次に来るピース)
- 色解放ドット (5 つ) + `X/20` カウンタ

## 想定パラメータ (JSX 由来)

| 名前 | 値 | 意味 |
|---|---|---|
| ドロップクールダウン | ~0.45 s | 連射防止 |
| `BURST_LERP` | 0.18 | バースト時の伸長補間係数 |
| `BURST_RETURN_LERP` | 0.09 | バースト戻りの補間係数 |
| バースト戻り遅延 | 28 frames | ターゲットを 1.0 に戻すまで |
| バースト伸長倍率 | 1.5 〜 2.6x | 棒の伸び幅 (ランダム) |
| 爆風半径 | r × 2.4 | バースト時の衝撃波範囲 |
| 16x 消滅ボーナス | 350 pts | 16x 同士マージ時の追加スコア |
| レアサイズ確率 | 20% | レベル 1〜5 ランダム出現 |
| 色アンロック間隔 | 20 個 | パープル / イエローの解放 |

## 経緯 (プロンプトログ要旨)

1. **初版** — 物理マージゲーム + ↑ボタン照準 + 3 色 + 16x 消滅 + 30 個毎の色解放 (4 色 → 5 色) + クールダウン
2. **操作変更** — クリックではなくバーをドラッグ → 離した瞬間にドロップ (pointerCapture)
3. **倍率テキスト削除** — Claude 中央のサイズ数字を取り去り、 スターバースト形状のみに
4. **レアサイズ追加** — 20% で 1〜5 ランダムサイズ、 大きいのが突然降る
5. **色解放を 30 → 20 個に変更** — 20/40 でパープル/イエロー解放
6. **バースト演出** — ゲームエリア内の Claude をタップで棒が伸縮 + 周囲に衝撃波 (物理にも影響)
